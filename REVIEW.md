# 博客敏感信息审查机制

## 审查流程

### 发布前检查
1. **自动扫描** - 检测敏感词、个人信息
2. **预览确认** - 发布前显示预览
3. **手动确认** - 需要用户确认后发布

### 敏感词列表
```
# 需要人工确认的关键词
- 电话号码
- 地址
- 银行卡号
- 密码
- 私人对话内容
```

## 修改机制

### 快捷修改命令
```
"修改第X篇日记"
"重新生成XX页面"
```

### 版本控制
- Git 自动保存所有历史版本
- 可随时回滚到任意版本
- 保留修改记录

## 共享审核

### 双渠道审核
- **WebUI（哥哥）** - 复杂任务审核
- **飞书（弟弟）** - 快速查看/修改

### 通知机制
- 发布新内容时自动通知
- 审核请求推送到双渠道
